ActionClient:
/go_to_point (rt2_assignment1::PoseAction)

Description:

This node communicates with the action server for go_to_point, issuing new goals whenever a 'start' request is received from the user_interface. At the same time it can stop a running goal if a request for 'stop' is received from the user_interface.
