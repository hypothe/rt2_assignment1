Description\+:

This node communicates with the action server for \mbox{\hyperlink{namespacego__to__point}{go\+\_\+to\+\_\+point}}, issuing new goals whenever a \textquotesingle{}start\textquotesingle{} request is received from the \mbox{\hyperlink{namespaceuser__interface}{user\+\_\+interface}}. At the same time it can stop a running goal if a request for \textquotesingle{}stop\textquotesingle{} is received from the \mbox{\hyperlink{namespaceuser__interface}{user\+\_\+interface}}.
